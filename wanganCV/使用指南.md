## 人脸检测

### 方法一：命令行工具（推荐）

无需启动服务器，直接运行脚本。

#### 基本用法

```bash
# 检测单张图片
python -m app.detect_faces image.jpg
```

**输出示例：**
```
图片尺寸: 1280 x 720
检测到人脸数量: 1

人脸信息:
  人脸 1:
    位置: (320, 120) -> (620, 520)
    尺寸: 300 x 400
    置信度: 0.9876
```

#### 保存裁剪的人脸

```bash
python -m app.detect_faces image.jpg --save-crops --output-dir outputs/faces
```

**参数说明：**
- `--save-crops`: 保存检测到的人脸裁剪图片
- `--output-dir`: 指定输出目录（默认: `outputs/faces`）
- `--margin-ratio`: 裁剪时的边距比例（默认: 0.2，即扩展 20%）

**输出示例：**
```
图片尺寸: 1280 x 720
检测到人脸数量: 1

人脸信息:
  人脸 1:
    位置: (320, 120) -> (620, 520)
    尺寸: 300 x 400
    置信度: 0.9876

保存的裁剪图片:
  人脸 1: C:\wanganCV\outputs\faces\faces_20250108_140530_1.jpg
```

#### 计算 Embedding

```bash
python -m app.detect_faces image.jpg --embeddings
```

**输出示例：**
```
图片尺寸: 1280 x 720
检测到人脸数量: 1

人脸信息:
  ...

Embedding 数量: 1
  人脸 1 embedding 维度: 128
```

#### 保存 JSON 结果

```bash
python -m app.detect_faces image.jpg --output-json result.json
```

生成的 JSON 文件格式：
```json
{
  "image": {
    "width": 1280,
    "height": 720
  },
  "num_faces": 1,
  "faces": [
    {
      "box": {
        "x1": 320,
        "y1": 120,
        "x2": 620,
        "y2": 520,
        "width": 300,
        "height": 400
      },
      "score": 0.9876
    }
  ],
  "saved": [],
  "embeddings": []
}
```

#### 调整置信度阈值

```bash
# 提高阈值，只检测高置信度的人脸
python -m app.detect_faces image.jpg --conf-threshold 0.7

# 降低阈值，检测更多人脸
python -m app.detect_faces image.jpg --conf-threshold 0.3
```

#### 完整示例

```bash
# 检测 + 保存裁剪 + 计算 embedding + 保存 JSON
python -m app.detect_faces image.jpg \
  --save-crops \
  --output-dir outputs/faces \
  --margin-ratio 0.2 \
  --embeddings \
  --output-json result.json \
  --conf-threshold 0.5
```

#### 所有命令行参数

| 参数 | 说明 | 默认值 |
|------|------|--------|
| `image` | 图片文件路径（必需） | - |
| `--conf-threshold` | 置信度阈值 | 0.5 |
| `--save-crops` | 保存裁剪的人脸 | False |
| `--output-dir` | 裁剪图片输出目录 | `outputs/faces` |
| `--margin-ratio` | 裁剪边距比例 | 0.2 |
| `--embeddings` | 计算 embedding | False |
| `--output-json` | 保存 JSON 结果 | None |
| `--quiet` | 静默模式 | False |

#### 调用 API

**使用 curl：**
```bash
# 基本检测
curl -X POST "http://127.0.0.1:8000/detect" \
  -F "file=@image.jpg"

# 保存裁剪 + 计算 embedding
curl -X POST "http://127.0.0.1:8000/detect?save_crops=true&return_embeddings=true" \
  -F "file=@image.jpg"
```

**使用 Python requests：**
```python
import requests

url = "http://127.0.0.1:8000/detect"
files = {"file": open("image.jpg", "rb")}
params = {
    "conf_threshold": 0.5,
    "save_crops": True,
    "return_embeddings": True
}

response = requests.post(url, files=files, params=params)
result = response.json()
print(result)
```

## 批量提取 Embedding

用于从注册清单中批量提取人脸特征向量。

### 准备注册清单文件

创建 `data/templates/manifest_enroll.csv` 文件，格式如下：

```csv
user_id,filepath,timestamp,qc_status
user_001,outputs/faces/face1.jpg,2025-01-08T10:00:00Z,ok
user_001,outputs/faces/face2.jpg,2025-01-08T10:05:00Z,ok
user_002,outputs/faces/face3.jpg,2025-01-08T10:10:00Z,ok
```

**字段说明：**
- `user_id`: 用户 ID
- `filepath`: 图片文件路径（相对或绝对路径）
- `timestamp`: 时间戳（可选）
- `qc_status`: 质量控制状态，只有 `ok` 的记录会被处理

### 运行提取脚本

```bash
# 使用默认参数
python -m app.extract_embeddings

# 指定自定义路径
python -m app.extract_embeddings \
  --manifest data/templates/manifest_enroll.csv \
  --output data/templates \
  --index data/templates/emb_index.csv \
  --margin 0.2
```

**参数说明：**
- `--manifest`: 注册清单文件路径（默认: `data/templates/manifest_enroll.csv`）
- `--output`: 输出根目录（默认: `data/templates`）
- `--index`: embedding 索引文件路径（默认: `data/templates/emb_index.csv`）
- `--margin`: 裁剪 margin 比例（默认: 0.2）

### 输出结果

**目录结构：**
```

│   ├── user_001/
│   │   ├── emb_0001_raw.npy      # 原始 embedding
```

## 创建身份模板

将同一用户的多张 embedding 聚合成一个稳定的身份模板。

### 基本用法

```bash
# 使用默认方法（简单平均）
python -m app.template_aggregator
```

### 选择聚合方法

```bash
# 方法 1: 简单平均（推荐，默认）
python -m app.template_aggregator --method mean

# 方法 2: 中位数（对异常值更鲁棒）
python -m app.template_aggregator --method median

# 方法 3: 加权平均（需要自定义权重）
# 注意：当前版本需要修改代码来提供权重
```

### 自定义路径

```bash
python -m app.template_aggregator \
  --embs-dir data/templates/embs \
  --output data/templates/templates \
  --method mean
```

**参数说明：**
- `--embs-dir`: embedding 基础目录（默认: `data/templates/embs`）
- `--output`: 模板输出目录（默认: `data/templates/templates`）
- `--method`: 聚合方法（`mean`, `median`, `weighted_mean`，默认: `mean`）
- `--use-raw`: 使用原始 embedding（默认使用归一化版本）

### 输出结果

**目录结构：**
```
data/templates/
└── templates/
    ├── template_user_001.npy
    ├── template_user_002.npy
    └── ...
```

**运行输出示例：**
```
找到 2 个用户，开始创建模板...
成功: 用户 user_001 - 5 个 embedding -> data/templates/templates/template_user_001.npy
成功: 用户 user_002 - 3 个 embedding -> data/templates/templates/template_user_002.npy

处理完成: 总计=2, 成功=2, 失败=0
```

### 聚合方法说明

1. **mean（简单平均）**
   - 计算所有 embedding 的平均值
   - 优点：简单高效，对噪声有鲁棒性
   - 适用：大多数场景，推荐使用

2. **median（中位数）**
   - 计算所有 embedding 的中位数
   - 优点：对异常值更鲁棒
   - 适用：数据质量不稳定，存在异常样本时

3. **weighted_mean（加权平均）**
   - 根据权重计算加权平均
   - 优点：可以根据样本质量分配权重
   - 适用：需要区分样本重要性时（需要修改代码提供权重）

### 在代码中使用

```python
from app.template_aggregator import (
    load_user_embeddings,
    aggregate_embeddings,
    create_user_template
)

# 方法 1: 加载并聚合
embeddings = load_user_embeddings("data/templates/embs/user_001", use_normalized=True)
template = aggregate_embeddings(embeddings, method="mean")
print(f"模板维度: {template.shape}, L2 范数: {np.linalg.norm(template)}")

# 方法 2: 直接创建模板
result = create_user_template(
    user_id="user_001",
    user_emb_dir="data/templates/embs/user_001",
    output_dir="data/templates/templates",
    method="mean"
)
```

## 验证模板质量

验证聚合模板与原始 embedding 的相似度，评估模板质量。

### 基本用法

```bash
# 验证所有模板
python -m app.verify_template
```

### 自定义路径

```bash
python -m app.verify_template \
  --templates-dir data/templates/templates \
  --embs-dir data/templates/embs
```

**参数说明：**
- `--templates-dir`: 模板目录（默认: `data/templates/templates`）
- `--embs-dir`: embedding 基础目录（默认: `data/templates/embs`）
- `--use-raw`: 使用原始 embedding（默认使用归一化版本）

### 输出示例

```
找到 2 个模板，开始验证...

用户 user_001:
  Embedding 数量: 5
  与模板相似度:
    平均: 0.9757
    最小: 0.9636
    最大: 0.9825
    标准差: 0.0064

用户 user_002:
  Embedding 数量: 3
  与模板相似度:
    平均: 0.9689
    最小: 0.9542
    最大: 0.9781
    标准差: 0.0102

汇总统计:
  平均相似度: 0.9723
  最小相似度: 0.9689
  最大相似度: 0.9757
```

### 相似度指标说明

- **平均相似度 > 0.95**：模板质量优秀
- **平均相似度 0.90-0.95**：模板质量良好
- **平均相似度 < 0.90**：模板质量较差，建议检查数据

- **标准差 < 0.01**：模板稳定性很好
- **标准差 0.01-0.02**：模板稳定性良好
- **标准差 > 0.02**：模板稳定性较差，建议增加样本数量

```python
import glob

photos = glob.glob("user_photos/user_003_*.jpg")
```

```bash

  --manifest data/templates/manifest_enroll.csv
```

```bash
python -m app.template_aggregator --method mean
```

#### 步骤 6: 验证模板质量

```bash
python -m app.verify_template
```

#### 步骤 7: 使用模板进行身份识别

```python
import numpy as np
from app.verify_template import cosine_similarity
from app.face_detector import detect_faces_from_path, compute_embeddings
import cv2

# 加载用户模板
template = np.load("data/templates/templates/template_user_003.npy")

# 检测新图片中的人脸
image = cv2.imread("test_photo.jpg")
faces = detect_faces_from_path("test_photo.jpg")
embeddings = compute_embeddings(image, faces)

# 计算相似度
if embeddings:
    test_embedding = np.array(embeddings[0]['embedding'])
    similarity = cosine_similarity(template, test_embedding)
    
    # 设置阈值（通常 0.6-0.7）
    threshold = 0.65
    if similarity >= threshold:
        print(f"身份匹配！相似度: {similarity:.4f}")
    else:
        print(f"身份不匹配。相似度: {similarity:.4f}")
```

## 常见问题

### Q1: 首次运行提示模型文件不存在？

**A:** 系统会自动下载模型文件到 `models/` 目录。如果下载失败，可以手动下载：
- `deploy.prototxt`: https://raw.githubusercontent.com/opencv/opencv/master/samples/dnn/face_detector/deploy.prototxt
- `res10_300x300_ssd_iter_140000.caffemodel`: https://raw.githubusercontent.com/opencv/opencv_3rdparty/dnn_samples_face_detector_20170830/res10_300x300_ssd_iter_140000.caffemodel

### Q2: 检测不到人脸？

**A:** 尝试以下方法：
1. 降低置信度阈值：`--conf-threshold 0.3`
2. 检查图片质量（清晰度、光照）
3. 确保人脸在图片中清晰可见

### Q3: Embedding 提取失败？

**A:** 检查：
1. 是否安装了 `face-recognition`：`pip install face-recognition`
2. 图片中是否检测到人脸
3. 图片路径是否正确

### Q4: 模板相似度较低？

**A:** 可能原因：
1. 样本数量太少（建议至少 3-5 张）
2. 样本质量差异大（光照、角度、表情）
3. 建议增加样本数量或提高样本质量

### Q5: 如何批量处理多张图片？

**A:** 使用 Python 脚本：
```python
import glob
import subprocess

images = glob.glob("photos/*.jpg")
for img in images:
    subprocess.run([
        "python", "-m", "app.detect_faces",
        img,
        "--save-crops",
        "--output-dir", "outputs/faces"
    ])
```

### Q6: 如何提高识别准确率？

**A:** 建议：
1. 使用更多样本（5-10 张）
2. 确保样本多样性（不同角度、光照、表情）
3. 使用 `mean` 方法聚合模板
4. 验证模板质量，确保平均相似度 > 0.95

### Q7: 支持哪些图片格式？

**A:** 支持 JPEG、PNG、BMP、WebP 格式。

### Q8: 如何在不同用户间进行身份识别？

**A:** 示例代码：
```python
import numpy as np
from app.verify_template import cosine_similarity
from app.face_detector import detect_faces_from_path, compute_embeddings
import cv2
import glob

# 加载所有用户模板
templates = {}
for template_file in glob.glob("data/templates/templates/*.npy"):
    user_id = template_file.split("_")[-1].replace(".npy", "")
    templates[user_id] = np.load(template_file)

# 检测新图片
image = cv2.imread("test.jpg")
faces = detect_faces_from_path("test.jpg")
embeddings = compute_embeddings(image, faces)

if embeddings:
    test_emb = np.array(embeddings[0]['embedding'])
    
    # 与所有模板比较
    best_match = None
    best_similarity = 0
    threshold = 0.65
    
    for user_id, template in templates.items():
        similarity = cosine_similarity(template, test_emb)
        print(f"{user_id}: {similarity:.4f}")
        
        if similarity > best_similarity:
            best_similarity = similarity
            best_match = user_id
    
    if best_similarity >= threshold:
        print(f"识别结果: {best_match} (相似度: {best_similarity:.4f})")
    else:
        print("未识别到匹配用户")
```
